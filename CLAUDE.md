# Lettre Pro — CLAUDE.md

## Projet
Site gratuit de modèles de lettres. Les utilisateurs copient un modèle, personnalisent les champs entre crochets et envoient.

## Stack
- Next.js 14 (App Router), TypeScript
- Tailwind CSS (couleurs `pro-*` = bleu, `accent-*` = vert)

## Services
| Service | Prix | Route |
|---|---|---|
| Lettre de motivation | Gratuit | /motivation |
| Lettre officielle | Gratuit | /officielle |
| Modèles gratuits | Gratuit | /modeles |

## Monétisation
AdSense (ca-pub-5064203547863113)
